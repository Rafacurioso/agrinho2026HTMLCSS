# 🌱 AgroCarbono Master

Sistema web interativo desenvolvido para o **Concurso Agrinho 2026**, com o objetivo de conscientizar sobre sustentabilidade no agronegócio por meio de ferramentas digitais, cálculos ambientais e conteúdos educativos.

---

## 📋 Sobre o Projeto

O **AgroCarbono Master** demonstra como a tecnologia pode auxiliar produtores rurais na tomada de decisões mais sustentáveis, promovendo a redução dos impactos ambientais e incentivando práticas alinhadas ao desenvolvimento sustentável.

O sistema reúne calculadoras, planejamento sustentável e um quiz educativo em uma interface moderna, responsiva e com modo claro/escuro.

---

## 🚀 Funcionalidades

### ☁️ Calculadora de Emissões de Carbono

Permite estimar as emissões anuais de gases de efeito estufa com base em:

- Consumo de diesel;
- Quantidade de bovinos no rebanho.

O sistema calcula o total de emissões e sugere a quantidade aproximada de hectares em sistema ILPF necessária para compensação ambiental.

---

### 💧 Pegada Hídrica

Calcula o consumo estimado de água considerando:

- Tipo de cultivo;
- Área irrigada.

Também apresenta uma dica sobre o uso de sensores de umidade do solo para reduzir o desperdício de água.

---

### 📈 Planejador Inteligente

Gera recomendações sustentáveis personalizadas com base em:

- Orçamento disponível;
- Tamanho da propriedade.

Entre as sugestões estão:

- Compostagem;
- Rotação de culturas;
- Irrigação por gotejamento;
- Bioinsumos;
- Sistema ILPF;
- Energia solar.

---

### 🧠 Quiz Educativo

O projeto possui um quiz interativo com perguntas sobre:

- Constituição Federal e meio ambiente;
- Emissões de gases do efeito estufa;
- Tecnologias sustentáveis no agronegócio.

Ao final, o usuário recebe automaticamente sua pontuação.

3. Abra o arquivo `index.html` em um navegador.
4. Use as ferramentas disponíveis no painel do sistema.
